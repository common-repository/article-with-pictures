如果文章没有缩略图，可以通过本插件自动给文章生成缩略图，同时支持在文章内容页显示缩略图。

== Description ==

支持文章自动生成缩略图、在文章内容页显示缩略图、将缩略图永久更新至文章。

如果文章有缩略图，则会显示文章原缩略图。只有文章没有缩略图，才会显示插件自动生成的缩略图。

如果文章内容没有图片，可以配置是否在内容页显示缩略图，让文章图文并茂。

只有主题支持缩略图功能，才会显示缩略图。

1、支持设置缩略图背景颜色（多个）、文字颜色（多个）、文字大小、文字字体、高度、宽度等。

2、插件生成的缩略图为特色图片，即使卸载了插件也可以依旧使用缩略图。

3、支持文章内容显示缩略图，可在文章首段、尾段、中间、随机显示。

4、可以设置在缩略图上是否添加文章标题文字。

5、支持使用多张背景图片（或者下载好的图片）来生成缩略图。（商业版）

6、支持使用Pixabay网站上的图片来生成缩略图，百万图片免费使用。（商业版）

== Installation ==

1. 进入WordPress网站后台，找到“插件-安装插件”菜单；
2. 点击界面左上方的“上传插件”按钮，选择本地提前下载好的插件压缩包文件（zip格式），点击“立即安装”；
3. 安装完成后，启用 “文章配图” 插件；
4. 通过“设置”链接进入插件设置界面；
5. 完成设置后，插件就安装完毕了。

== Frequently Asked Questions ==

= 使用文章配图后有什么效果？ =
让网站页面显示更友好，同时对收录有一定的帮助。

= 为什么缩略图上没有文字？ =
确认是否启用了“添加缩略图文字”功能，然后确认是否选择了“缩略图文字字体”。

= 为什么缩略图文字字体无法选择？ =
安装插件后，需要自行下载中文字体文件(后缀为.ttf文件)，将字体文件上传至/wp-content/plugins/article-with-pictures/fonts目录下。注意：字体文件的名称不能有空格等特殊符号，建议使用拼音重新命名字体文件，例如：ziti1.ttf、ziti2.ttf。

= 为什么没有显示缩略图？ =
首先确认自己的主题是否支持缩略图功能，如果支持，尝试在文章配图设置中将主动生成特色图片设置为是。如果依旧不行，请联系作者处理。

= 为什么生成的缩略图文字模糊？ =
这与缩略图的高度与宽度、字体文件、字体大小等有关，调整相关设置以达到完美显示即可。

= 生成的缩略图在哪里查看、哪里删除？ =
可以在网站后台-媒体-媒体库页面看到或删除。

= 卸载插件后，是否会删除缩略图？ =
卸载插件后，除生成的缩略图不会删除，其余内容均会删除。

= 为什么文章配图插件无法正常使用？ =
如果您装的其它插件或者主题对缩略图也进行了处理，则文章配图插件很有可能无法正常运行。

= 商业版有哪些功能？ =
商业版支持使用背景图片、Pixabay网站上的图片来生成缩略图。

= 缩略图背景图片无法设置成功？ =
免费版无此功能。商业版用户如果设置不成功，很有可能是您的图片格式问题，您可以尝试使用png格式的背景图。
